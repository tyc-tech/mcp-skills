# 投资机构画像

## 触发条件

当用户意图涉及以下场景时触发本 Skill：

- 了解某投资机构的投资偏好、规模和历史
- 创业者寻找融资时研究目标投资机构
- FA（财务顾问）为项目匹配合适的投资机构
- 用户提到"投资机构"、"机构画像"、"VC/PE分析"、"投资人背景"等关键词

## 输入要求

| 参数 | 是否必填 | 说明 |
|------|---------|------|
| name | 必填 | 投资机构名称（即公司名称或机构名称） |

> 注意：本 Skill 中 oi 系列接口使用 name 参数（而非 keyword），Step 1 使用 keyword 参数。name 值与 keyword 值相同，均为机构名称。

## 执行流程

### Step 1: 基本信息

- 调用 `t1_ic_baseinfoV3` (参数: keyword=机构名称) -- 获取投资机构的工商基本信息，包括注册资本、法定代表人、成立日期等

### Step 2: 投资事件

- 调用 `t1_oi_businessEvent` (参数: name) -- 获取被投资企业信息，包括企业名称、投资比例等

### Step 3: 公开投资产品

- 调用 `t1_oi_publicInvest` (参数: name) -- 获取公开投资产品信息，包括产品名称、所属公司、参与轮次、投资时间、投资金额等

### Step 4: 非公开投资

- 调用 `t1_oi_secretInvest` (参数: name) -- 获取未公开投资企业信息，包括企业名称、法人、注册资本、成立日期等

### Step 5: 机构团队

- 调用 `t1_oi_team` (参数: name) -- 获取机构成员信息，包括姓名、职位、简介等

### Step 6: 旗下基金

- 调用 `t1_oi_funds` (参数: name) -- 获取基金管理公司的企业信息，包括企业名称、法定代表人、注册资本、成立日期等

### Step 7: 投资统计分析

- 调用 `t1_oi_stats` (参数: name) -- 获取投资机构投资统计分析，包括投资数量、投资规模、投资行业分布等

### Step 8: 相关新闻

- 调用 `t1_oi_news` (参数: name) -- 获取投资机构相关新闻舆情信息，包括发稿媒体、标题、发布时间等

## 错误处理

- 任一 Tool 调用返回 `error_code != 0`，记录错误信息，在报告中标注"该项查询失败，建议人工复核"
- 如 Step 1 企业基本信息查询失败（企业不存在），直接终止流程并提示"未查询到该机构信息，请核实机构名称"
- 如 Step 2 投资事件为空，标注"未查询到该机构的投资事件记录"
- 如遇"服务暂时不可用"（熔断），在报告中标注"该项服务暂不可用，建议稍后重试"
- 部分步骤失败不影响其他步骤执行，最终报告中标注缺失项

## 示例

**用户输入：**
> 帮我分析"某某创投"的投资机构画像

**执行过程：**
1. 调用 `t1_ic_baseinfoV3`(keyword="某某创投") -> 成立于2010年，注册资本 5 亿
2. 调用 `t1_oi_businessEvent`(name="某某创投") -> 累计投资 56 家企业
3. 调用 `t1_oi_publicInvest` -> 公开投资产品 23 个
4. 调用 `t1_oi_secretInvest` -> 非公开投资 18 家
5. 调用 `t1_oi_team` -> 核心团队 8 人
6. 调用 `t1_oi_funds` -> 旗下 5 只基金
7. 调用 `t1_oi_stats` -> 年均投资 8 笔，偏好科技/医疗赛道
8. 调用 `t1_oi_news` -> 近期 3 条投资新闻

**输出：** 生成投资机构画像报告，该机构专注早中期科技/医疗投资，管理规模中等，投资活跃。
