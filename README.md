## Skill 命名规范

- 命令前缀: `/tyc-`
- 格式: `/tyc-<场景英文简称>`
- 示例: `/tyc-kyb`、`/tyc-ic-memo`、`/tyc-listed`

## Skill 文件结构

每个 Skill 目录包含一个 `SKILL.md`，结构如下：

```
mcp_skills/<category>/<skill-name>/SKILL.md
```
